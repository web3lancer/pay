## Tech Stack

- **Next.js 15** + **React 19**
- **Appwrite** (Auth, DB, Storage, Realtime)
- **Tailwind CSS** + **Framer Motion**
- **TypeScript**
- **Zustand** (state management)
- **Recharts** (visualizations)
- **Dexie** (IndexedDB)
